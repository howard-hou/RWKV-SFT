# RWKV-SFT

This is a simple RWKV-7 SFT code repository. old RWKV-6 SFT code has been moved to Dir RWKV-v6

## Installation

To use this code, you need to have the following dependencies installed:

- python 3.10
- torch 2.0.0+
- cuda 12.1+
- pytorch-lightning==1.9.5
